ссылка на типы вопросов: https://docs.ispring.ru/pages/viewpage.action?pageId=1444703
числовой вопрос: тот же самый TextQuestion, только валидация на числа.
ранжирование (Drag&Drop): Респондент должен расставить пункты в порядке предпочтений с помощью перетягиваний элементов.
соответствие: можно реализовать с помощью Drag&Drop, либо с выпадающим списком, исключая из сиска каждый выбранный ответ при выборе его в предыдщем выпадающем списке.
вложенные ответы: возле каждого вороса будет выпадающий список, в котором будет несколько вариантов ответов.
пропуски: Респонденту нужно заполнять пропуски в тексте.
вставьте пропущенные (Drag&Drop): Респондент должен заполнить пробелы в тексте, используя предоставленный банк слов с помощью перетягивания.
